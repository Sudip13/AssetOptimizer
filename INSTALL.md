# Installation Instructions

## Quick Install

1. **Download** the latest release or clone this repository
2. **Compress** the entire `AssetOptimizer` folder into a ZIP file
3. Open **Blender** (3.0 or later)
4. Go to `Edit > Preferences > Add-ons`
5. Click `Install...` button
6. Select the ZIP file
7. Enable **VR Asset Optimizer** checkbox

## Manual Install

### Windows
```
Copy AssetOptimizer folder to:
%APPDATA%\Blender Foundation\Blender\4.2\scripts\addons\
```

### macOS
```bash
cp -r AssetOptimizer ~/Library/Application\ Support/Blender/4.2/scripts/addons/
```

### Linux
```bash
cp -r AssetOptimizer ~/.config/blender/4.2/scripts/addons/
```

## Verify Installation

1. In Blender, press `N` to open the sidebar
2. Look for the **VR Assets** tab
3. You should see the VR Asset Optimizer panel

## First Use

1. Select any mesh object
2. Open VR Assets panel (`N` key)
3. Try the "CAD Import" preset
4. Review the statistics displayed
